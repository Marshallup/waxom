1 команда npm i - для установки всех модулей
2 команда - gulp - для запуска
При установке сторонних библиотек через npm - команда в конце '--save' без '-dave'

Bower:
bower init - через стандрантую виндовс консоль
bower info "бибилотека" - чтобы поискать бибилотеки
bower i -S - установить в dependenses(проект зависит от установленной бибилотеки)
bower i -D - установить в Devdependenses(проект не зависит от установленной бибилотеки)
bower uninstal "библиотека"
